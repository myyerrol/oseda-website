## 使用

### 下载仓库

```sh
$> cd your-workspaces
$> git clone git@github.com:oscc-web/oseda-website.git
```

### 更新依赖

```sh
$> ./setup.sh
```

### 运行网站

```sh
$> npm run command -- dev
```

### 更新仓库

```sh
$> git pull
```
